# jlspot 

Wrapping [Spot]() using CxxWrap.jl 

## How is this done

Download, extract and build spot source code
```bash 
wget http://www.lrde.epita.fr/dload/spot/spot-2.9.7.tar.gz
tar -xzf spot-2.9.7.tar.gz
rm spot-2.9.7.tar.gz
cd spot-2.9.7
./configure --disable-python
```
